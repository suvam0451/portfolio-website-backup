This starter supports [MDX](https://mdxjs.com). MDX is a format that lets you

<Counter />

seamlessly use JSX in your Markdown documents. You can import components, like

interactive charts or notifs, and export metadata.
Find the above React component in `/src/components/Counter.tsx`.

![](images/riverScene.jpg)

```jsx{1,4-6}
import React from "react";

const Counter = initialCounter => {
	const [counter, setCounter] = React.useState(initialCounter);

	const onIncrement = () => {
		setCounter(c => c + 1);
	};

	const onIncrement = () => {
		setCounter(c => c - 1);
	};

	return (
		<div>
			{counter}

			<div>
				<button onClick={onIncrement} type="button">
					Increment
				</button>
				<button onClick={onDecrement} type="button">
					Decrement
				</button>
			</div>
		</div>
	);
};

export default Counter;
```
